How It Works

This demo uses CSS for the parallax effect and JavaScript to handle the theme switching functionality.

The Parallax Effect

The parallax effect is achieved using CSS's background-attachment: fixed property. This makes the background image stay in place while the content scrolls over it.

For the best effect, use high-resolution images and ensure there's enough contrast between the background and text.

Theme Switching

The theme switching is handled using CSS variables and a small JavaScript function that toggles a data-theme attribute on the document element.

When the dark theme is active, we change the background image and update all the color variables for a cohesive look.
