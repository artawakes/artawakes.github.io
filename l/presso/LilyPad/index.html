The problem

Five forces holding the system back

01 Third-party lock-in Mantine under every component.
02 Two sources of truth Figma and code drift; neither is canonical.
03 Not machine-readable No spec an AI can implement from.
04 Slow net-new work Every component re-litigates the same decisions.
05 No sanctioned extension path No consistent way to extend the system.
