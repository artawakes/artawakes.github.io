Lyft Home Search

Reimagining the search experience on Lyft's homepage to drive user engagement and increase ride bookings.

UX Design
Search Experience
Conversion Optimization

The Challenge

Lyft's homepage search functionality needed enhancement to:

  • Improve the speed and accuracy of location searches
  • Reduce user friction in the booking process
  • Increase conversion rates for ride bookings
  • Enhance the mobile search experience
  • Integrate with real-time location data

Research & Discovery

User Research

  • User behavior analysis
  • Search pattern study
  • Usability testing
  • Competitor analysis

Technical Analysis

  • Performance metrics
  • Search algorithm review
  • API integration assessment
  • Mobile optimization study

Solution & Implementation

  • Redesigned search interface with predictive suggestions
  • Implemented smart location detection
  • Enhanced mobile-first search experience
  • Integrated real-time pricing and ETA
  • Added favorite locations feature
  • Optimized search results ranking

Results & Impact

  • 35% increase in search completion rate
  • 45% reduction in search abandonment
  • 25% improvement in conversion rate
  • 60% faster average search time
  • 92% positive user feedback

Project Details

  • Role: Senior UX Designer
  • Duration: 6 months
  • Tools: Figma, Sketch, Principle
  • Team: 3 designers, 4 engineers

  • • Location detection
  • • Real-time pricing
  • • Mobile optimization
