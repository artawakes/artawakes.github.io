EA Design System

Leading the development of a comprehensive design system to unify Electronic Arts' digital products and improve design efficiency.

Design System
UX Design
Enterprise Design

Overview

Led the development and implementation of a cross-functional design system for Electronic Arts, focusing on creating a unified design language across multiple digital products and platforms. The project involved extensive collaboration with various teams to establish consistent design patterns and improve overall design efficiency.

Challenge

EA needed a unified design system that could:

  • Standardize design patterns across multiple products
  • Improve design and development efficiency
  • Ensure consistent user experience
  • Scale across different platforms and devices
  • Support rapid product development

Approach

Research & Analysis

  • Conducted audit of existing design patterns
  • Analyzed user interface inconsistencies
  • Gathered stakeholder requirements
  • Reviewed competitor design systems

Design & Development

  • Created core design principles and guidelines
  • Developed component library
  • Built documentation and usage guidelines
  • Implemented version control system

Implementation

  • Rolled out system in phases
  • Provided team training and support
  • Established governance model
  • Created maintenance procedures

Key Features

  • Comprehensive component library
  • Responsive design patterns
  • Accessibility guidelines
  • Interactive documentation
  • Design tokens and variables
  • Version control system

Impact & Results

  • 40% reduction in design-to-development time
  • 60% improvement in design consistency
  • Increased team collaboration
  • Streamlined design process
  • Enhanced product quality

Project Details

  • Role: Lead Designer
  • Duration: 18 months
  • Tools: Figma, Storybook, React
  • Team: Design, Engineering, Product
